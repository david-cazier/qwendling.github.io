Modèle topologique
  • Objectif : représenter la décomposition cellulaire de l'objet
  • Pouvoir parcourir les différentes cellules (sommets,arêtes,faces,volumes) de l'objet
  • Pouvoir parcourir le voisinnage de chaque cellule de manière efficace
  • Exemple : Les cartes combinatoire
    • Représente un objet de dimension n
    • Une entité de base : les brins
    • n relations entre les brins
    • Contraintes sur ces relations
